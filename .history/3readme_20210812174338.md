1. La création d'objet : on initialise un objet vide puis on lui donne ses propriétés
   
   ```js
   function creerAnimal(nom, age, sexe) {
    var animal = {};
    animal.nom = nom,
    animal.age = age,
    animal.sexe = sexe
    return animal;
}
   ```
